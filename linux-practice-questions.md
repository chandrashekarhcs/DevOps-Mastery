# Linux Practice Questions

## Day 2
1. **What is the command to list files in a directory?**  
   Answer: `ls`

2. **How do you create a new directory?**  
   Answer: `mkdir <directory-name>`

## Day 3
1. **What does the `chmod` command do?**  
   Answer: It changes the permissions of files or directories.

2. **How can you view the contents of a file?**  
   Answer: `cat <file-name>`

## Day 2 - Intro to Linux
1. **What is the purpose of the `pwd` command?**  
   Answer: It prints the current working directory.

2. **What is the difference between `scp` and `sftp`?**  
   Answer: `scp` is used for secure file copying over SSH, while `sftp` is an interactive file transfer program.

## OS Cloud Warmup
1. **What command is used to update package information on Debian-based systems?**  
   Answer: `sudo apt update`

2. **How do you check the current disk usage?**  
   Answer: `df -h`.

3. **Name any two common commands for managing services?**  
   Answer: `systemctl start <service>` and `systemctl stop <service>`.
